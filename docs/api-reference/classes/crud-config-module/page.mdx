# `CrudConfigModule`

NestJS module entrypoint providing `register()` and `registerAsync()` methods.

*Defined in: `src/modules/config/config.module.ts`*

## API

```ts
export class CrudConfigModule {
	static register(options: IConfigOptions): DynamicModule;
	static registerAsync(properties: ICrudConfigAsyncModuleProperties): DynamicModule;
}
```

## Notes

- `register()` is fully synchronous.
- `registerAsync()` requires compile-time options (controllers/entities) in `staticOptions`.
