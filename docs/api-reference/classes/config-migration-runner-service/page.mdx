# `ConfigMigrationRunnerService`

Low-level runner that executes individual migration definitions.

*Defined in: `src/modules/config/migration/migration-runner.service.ts`*

## API

```ts
export class ConfigMigrationRunnerService {
	onApplicationBootstrap(): Promise<void>;
	runMigrations(): Promise<void>;
}
```

## Notes

- Runs migrations on startup when `migrationOptions.isEnabled` and `migrationOptions.shouldRunOnStartup` are enabled.
