# `CrudConfigService`

Main programmatic API to read/write configuration values.

_Defined in: `src/modules/config/config.service.ts`_

## API

```ts
export class CrudConfigService {
 delete(options: IConfigDeleteOptions): Promise<void>;
 get(options: IConfigGetOptions): Promise<IConfigData>;
 getList(options: IConfigGetListOptions): Promise<IConfigData[]>;
 set(options: IConfigSetOptions): Promise<IConfigData>;
}
```
