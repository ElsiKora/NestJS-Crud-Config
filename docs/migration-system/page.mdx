# Migration System

NestJS CRUD Config includes a migration system to initialize and evolve configuration state on startup.

## Key features

- Automatic execution on startup
- Transaction support
- Environment-specific migrations
- Migration tracking with statuses (`EConfigMigrationStatus`)
- Rollback support
- Stuck migration cleanup
