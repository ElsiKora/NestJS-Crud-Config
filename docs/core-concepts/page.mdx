# Overview

Core concepts of **NestJS CRUD Config**:

- how the module is registered (`register` / `registerAsync`)
- how dynamic entities are created and exported via tokens
- how dynamic controllers are created (optional)
- how `CrudConfigService` reads/writes configuration values
- optional encryption and caching
